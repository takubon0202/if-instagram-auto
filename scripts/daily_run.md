このファイルをClaude Codeに渡して実行してください。

Claude Codeで以下を実行:
```

```

## 固定条件（編集不要）

### ブランド
- 名称: if(塾)
- 形式: オンラインのみ
- 対象: 小・中・高
- 実施: 月曜 17:00–18:00
- 料金: 月 11,000円（税込）
- 強み: AIで支援 / 個別最適 / 第三の居場所
- CTA: https://if-juku.net/

### B2Bメニュー
- C: 社内研修（生成AI/プロンプト/セキュリティ）
- D: プロンプト/ワークフロー設計代行
- E: Web制作（LP/フォーム/予約導線）

### 投稿設定
- 毎日3本: 09:00 / 12:30 / 20:00
- 比率: 塾2 : 企業1
- 形式: カルーセル or リール

### 制約
- 未成年の写真/動画は使わない
- 発達・不登校の断定表現禁止（診断/治療/治る等）
- 配慮の一文を必ず入れる

## 実行手順（サブエージェント順）

### Step 1: Trend Scout
曜日に応じたテーマを決定し、3投稿の方向性を定める。

**週次テーママッピング:**
| 曜日 | 塾09:00 | 企業12:30 | 塾20:00 |
|------|---------|-----------|---------|
| 月 | 安心・居場所 | 研修の失敗と対策 | 安心・共感 |
| 火 | 学習のハードル | ローテーション | 小さく始める |
| 水 | 保護者向け | ワークフロー設計 | 声かけ・家庭 |
| 木 | AI/ITスキル | ローテーション | 小中高別 |
| 金 | 無料体験推し | LP改善チェック | 背中押し |
| 土 | FAQ | ローテーション | FAQ |
| 日 | まとめ | ローテーション | まとめ |

### Step 2: Planner
3投稿の企画を立案（タイトル、構成、ターゲットアクション）

### Step 3: Carousel Builder / Reel Scriptwriter
- 09:00, 12:30 → Carousel Builder（7-9枚構成）
- 20:00 → Reel Scriptwriter（15秒台本）

### Step 4: JP Copywriter
キャプション（120-250字）、ハッシュタグ（8-15個）、配慮文を作成

### Step 5: Anime Visual Prompt Engineer
各スライド/サムネイル用の画像プロンプトを生成

### Step 6: Safety & Policy QA
禁止表現、断定、煽り、個人情報をチェック→修正

### Step 7: Data Engineer
posts.jsonへ3件追記、検証実行
